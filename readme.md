

# 📌 Coding Challenge 05 – Payroll Processing Toolkit

## Overview

This project is a JavaScript payroll system that calculates employee pay using modular functions. It demonstrates function creation, DRY principles, and array iteration.

## Features

* Stores employees in an array of objects
* Calculates:

  * Base pay (up to 40 hours)
  * Overtime pay (1.5× rate over 40 hours)
  * Taxes (15%)
  * Gross and net pay
* Processes payroll using reusable functions
* Loops through employees with `.forEach()`

## How to Run

1. Open `index.html` in a browser
2. Open the Console
3. View payroll results

## Concepts Used

* Functions
* Pure functions
* Callbacks
* Array iteration
